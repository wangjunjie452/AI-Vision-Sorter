【*】程序简介

-工程名称		：DWT应用与DHT11温湿度传感器
-实验平台  		：野火 STM32F103C8T6-STM32开发板  

【 ！】功能简介：

上位机定期查看上传的温湿度数据，通过改变DHT11周围的温湿度环境，来变化采集数据

【 ！】实验操作：

	1.保证开发板相关硬件连接正确，用USB线连接开发板的USB转串口跟电脑，
	2.在电脑端打开串口，调试助手并配置好相关参数：115200 8-N-1，
	3.编译并下载程序到开发板即可观察现象，此时串口调试助手即可收到开发板发过来的数据。
	4.使用上位机定期查看上传的温湿度数据，通过改变DHT11周围的温湿度环境，来变化采集数据

/*********************************************************************************************/

【*】 引脚分配


DEBUG_TX->USART1_TXD->PA9		PA9作为USART1的TXD	核心板默认使用 PA9 做 DEBUG_TX

DHT11_DATA->PB12				DHT11温湿度传感器数字电平输入，数据通过一定电平规则(协议)变化，传感器模块默认上拉

/*********************************************************************************************/

【*】程序描述：

< main.c >

	1.初始化USART、DHT11和DWT等配置
	2.构建dht11协议文本，定期处理数据并通过上位机呈现
