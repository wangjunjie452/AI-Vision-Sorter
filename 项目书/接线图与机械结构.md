### 1.3 关键注意事项

```
⚠  共地原则：STM32 GND、PCA9685 GND、电源 GND、TTL 模块 GND 必须全部连在一起
⚠  舵机供电：必须接外部 5V 电源，不能从 STM32 3.3V 取电（电流不够）
⚠  TX/RX 交叉：STM32 的 PA9(TX) 接 TTL 模块的 RXD，PA10(RX) 接 TXD
⚠  I2C 上拉：PCA9685 模块一般自带上拉电阻，不需要额外加
```

### 2.3 CH0 挡板结构详解

```
侧视图（三种状态）：

    90° 水平挡住（默认）：
                ┃挡板━┓
                ┃     ┃
              ┌─┃─────┃
              │ ┃     ┃  ← 物体停在上面
              │ ┃SG90 ┃
              └─┃─────┃
                ┃

    0° 向上抬起（B盒）：
              ┏━挡板
              ┃╱
              ┃╱
              ┃╱    ← 物体滑落
              ┃╱
            ┌─┃──
            │ ┃SG90
            └─┃──
              ┃

    180° 向下放行（到CH1）：
              ┃
              ┃
            ┌─┃──
            │ ┃SG90
            └─┃──
              ┃
              ┗━挡板 ← 物体从上面滑过，继续往下
              ╲
               ╲
                ↓ 到 CH1 转臂
```

### 2.4 CH1 转臂结构详解

```
俯视图（转臂旋转示意）：

    CH1 = 0°（向左）：      CH1 = 90°（中间）：     CH1 = 180°（向右）：

         转臂                      │转臂               转臂
        ╱                          │                    ╲
       ╱                           │                     ╲
      ╱                            │                      ╲
     ● 舵机轴                     ● 舵机轴               ● 舵机轴
     │                            │                      │
     ↓                            ↓                      ↓
    盒A                           盒B                    盒C


转臂做法：
  1. 取硬纸板条 10×2cm
  2. 一头粘上舵机配的十字摇臂（盒子里有）
  3. 摇臂套在舵机轴上（用螺丝固定）
  4. 另一头做一个小挡边（防止物体滑出去）

       小挡边（折起来的纸板）
          ┌─┐
          │ │
    ──────┤ ├──────→ 粘在舵机摇臂上
          │ │
          └─┘
```

### 2.5 组装步骤

```
第一步：准备材料
  ─ 硬纸板剪出底板、斜坡、挡板、转臂
  ─ 准备 3 个小盒子
  ─ 热熔胶枪预热

第二步：粘舵机到底板
  ─ CH0 舵机：粘在斜坡出口处，轴朝侧面（挡板上下翻转）
  ─ CH1 舵机：粘在底板中间偏下，轴朝上（转臂水平旋转）
  ─ 用热熔胶固定，等冷却

第三步：粘斜坡
  ─ 斜坡一头对准入口，另一头对准 CH0 挡板
  ─ 角度约 30°，太陡物体冲太快，太缓物体滑不动
  ─ 用热熔胶固定

第四步：粘挡板和转臂
  ─ 挡板粘在 CH0 舵机摇臂上
  ─ 转臂粘在 CH1 舵机摇臂上
  ─ 通电测试角度是否正确

第五步：放盒子
  ─ 盒A 在 CH1 转臂左侧
  ─ 盒B 在 CH0 挡板正下方
  ─ 盒C 在 CH1 转臂右侧
  ─ 用热熔胶固定在底板上

第六步：测试
  ─ 串口助手发指令，观察挡板和转臂动作
  ─ 手动放小物体，看能不能正确分拣
```

## 三、Day 2 到货后操作清单

```
  □ 1. 收到快递，检查器件数量和外观
  □ 2. PCA9685 接面包板，按接线清单连 STM32

  □ 9. 调整 OBJECT_ARRIVE_DELAY_MS 参数
  □ 10. 完成标志：物体正确落入对应盒子
```
